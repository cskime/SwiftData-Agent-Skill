# AGENTS.md

## Skills

A skill is a set of local instructions stored in `SKILL.md`.

### Available skills in this repository

- `swiftdata-expert-skill`: iOS 26+ SwiftData guidance for modeling, querying, context isolation, history tracking, migration, custom data stores, inheritance, performance, and testing.
  - Path: `swiftdata-expert-skill/SKILL.md`

## How to use this repository

1. Keep `swiftdata-expert-skill/SKILL.md` concise and trigger-focused.
2. Put detailed guidance in `swiftdata-expert-skill/references/`.
3. Avoid adding optional metadata folders unless explicitly needed.
4. Validate links and file paths whenever references are updated.
